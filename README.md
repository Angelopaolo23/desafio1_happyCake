Esta aplicacion corresponde al primer desafio del Modulo React II de Desafio Latam, especificamente a la unidad de React Router I, esta aplicacion busca aprender la utilizacion y beneficios de utilizar React Router y una SPA, y consiste en una pagina web donde tenemos una Navbar que nos permite movernos entre dos Views (Home y Contacto), tambien existe una tercera View (Notfound) que se utiliza en caso de una direccion distinta a la de las Views antes mencionadas.
